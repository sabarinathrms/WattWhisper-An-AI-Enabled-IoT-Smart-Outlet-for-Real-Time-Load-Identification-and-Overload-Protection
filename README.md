# WattWhisper-An-AI-Enabled-IoT-Smart-Outlet-for-Real-Time-Load-Identification-and-Overload-Protection
WattWhisper is an AI-enabled IoT smart outlet using ESP8266 and INA219 to monitor real-time power usage, identify appliances through current signature analysis, and provide automatic overload protection. It integrates cloud-based AI and Blynk for safe, efficient, and remote energy management.
